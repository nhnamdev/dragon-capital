# Dragon Capital

Landing page demo cho Dragon Capital - công ty quản lý quỹ.

## Tech stack

- HTML5, CSS3, Vanilla JavaScript
- Font: Manrope, Playfair Display (Google Fonts)
- Icons: Font Awesome 6

## Tính năng

- Hero section với floating card, dots animation, scroll cue
- Pioneer section với badge "30+ năm" và founder card
- Stats section với 4 con số đếm động (counter animation)
- Funds section giới thiệu 3 sản phẩm quỹ
- Form liên hệ với validate cơ bản
- Footer với thông tin liên hệ + social links
- Back-to-top button
- Fully responsive

## Cách chạy

Mở file `index.html` trực tiếp trong trình duyệt, hoặc dùng live server bất kỳ.
